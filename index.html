FlickCam

A light weight, easy to use camera application for Android that talks to Raspberry Pi

View project onGitHub

Welcome to the FlickCam Project.

This project is about combining Raspberry Pi and Android. The aim is to take the hardware components of Raspberry Pi and let them "talk" to a a sleek Android camera application.
